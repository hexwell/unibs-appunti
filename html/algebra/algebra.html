2.1 Definizione di Relazione

Si dice che R ⊆ A × B è una relazione tra due insiemi A e B.

Se:
C =

Si scrive:
b1 = C(a1)
b2 = C(a2)

2.1.1 Proprietà delle Relazioni

Totalità a Sinistra

Una relazione R tra A e B si dice ovunque definita (o totale a sinistra) se:

∀x ∈ A.∃y ∈ B : (x,y) ∈ R

Funzionalità

Una relazione R tra A e B si dice funzionale se:

∀x ∈ A.∃y ∈ B : (x,y) ∈ R ∃!y ∈ B : (x,y) ∈ R

2.2 Definizione di Funzione

Una relazione f si dice funzione se è funzionale e ovunque definita.

“Funzione” si riferisce alla terna: associazione di elementi, dominio e codominio, non solo all’associazione di elementi. Specificare solo un’associazione non definisce una funzione: occorre specificare anche dominio e codominio. Infatti, due funzioni che hanno una “stessa” associazione di elementi ma diverso dominio e/o diverso codominio sono funzioni diverse.

Si scrive:
f : A → B
dove A è il dominio di f e B è il codominio di f.

2.2.1 Proprietà delle Funzioni

Iniettività

Una funzione da A a B si dice iniettiva se:

∀x,x′∈ A.f(x) = f(x′) x = x′

Suriettività

Una funzione da A a B si dice suriettiva (o totale a destra) se:

∀y ∈ B.∃x ∈ A : y = f(x)

(equivalentemente: im(f) = codom(f))

Biiettività

Una funzione si dice biiettiva (o biiezione, o anche corrispondenza 1 a 1 o biunivoca) se è sia iniettiva che suriettiva.

Osservazione:

f : A → B è biiettiva =

A e B possono essere infiniti.

< ⇐⇒∃ una funzione iniettiva X → Y ∧ ∄ una biiezione X → Y .

2.2.2 Definizione di Immagine

L’insieme di tutti i valori di f : A → B valutata in ogni elemento di un insieme S ⊆ A si dice l’immagine di S tramite f:

f[S] = f(S) := ⊆ B

Imf = im(f) = f[A]

L’immagine del dominio di una funzione f tramite f si dice immagine di f.

Il valore di f : A → B valutata in x ∈ A si dice immagine di x tramite f.

im(f) ⊆ codom(f)

2.2.3 Definizione di Controimmagine

L’insieme degli elementi del dominio di una funzione f : A → B che f associa a tutti gli elementi di S si dice controimmagine, preimmagine o immagine inversa di S tramite f:

f-1[S] = f-1(S) := ⊆ A

2.3 Strutture Algebriche

2.3.1 Definizione di Operazione Binaria

Sia U un insieme. Si dice operazione binaria una funzione o : U × U → U.

2.3.2 Definizione di Struttura Algebrica

Una struttura algebrica è una n-upla data da un insieme ed una o più operazioni su di esso:

(U,o)

2.3.3 Definizione di Associatività

Si dice che *è associativa se ∀a,b,c ∈ A.a * (b * c) = (a * b) * c.

2.3.4 Definizione di Elemento Neutro

Sia (A,*) un insieme con una operazione binaria (magma):

* : A × A → A
(a,b) = a * b

Si dice che e ∈ A è un elemento neutro per * se:

∀a ∈ A.e * a = a * e = a

2.3.5 Definizione di Inverso, Inverso Destro e Inverso Sinistro

Se (X,*) ammette elemento neutro e si dice che ∀x ∈ X:

x′è inverso destro di x se ∃x′∈ X : x * x′ = e
x′′è inverso sinistro di x se ∃x′′∈ X : x′′* x = e
x′′′è inverso di x se x′′′è inverso destro di x ∧ x′′′è inverso sinistro x.

2.3.6 Definizione di Commutatività

Si dice che *è commutativa se ∀a,b ∈ A.a * b = b * a.

2.3.7 Definizione di Monoide

(A,*) (magma) è detto monoide se *è associativa (semigruppo) e ammette elemento neutro.

2.3.8 Definizione di Gruppo

(A,*) è detto gruppo se è un monoide ∧ ogni elemento di A ammette inverso (necessariamente unico, destro e sinistro, solitamente indicato con a-1).

2.3.9 Definizione di Gruppo Abelliano

(A,*) è detto gruppo abelliano o commutativo se oltre ad essere un gruppo, * è commutativa.

2.3.10 Definizione di Sottrazione

La sottrazione è definita come somma con l’opposto di un elemento in (ℤ,+).

a - b = a + (-b)

2.5.9 Rango

Definizione di Minore di Ordine p

Data una matrice A ∈ Matm,n(K) si dice minore di ordine p una matrice quadrata di ordine p ottenuta da A sopprimendo n - p colonne e m - p righe.

Definizione di Rango

Data una matrice A ∈ Matm,n(K) dire che il rango di A è p:

(
rg(A) = p
r(A) = p
ρ(A) = p
)

con p ≤ min(m,n)

significa dire che A ha un minore non singolare di ordine p, e che ogni eventuale minore di ordine p + 1 è singolare.

r(A) = 0 ⇐⇒ A = 0

Se A ∈ Matn(K) allora r(A) = n ⇐⇒≠0

A ha rango massimo = A ∈ Matn(K), r(A) = n

1 ≤ rg(A) ≤ min(m,n), A ∈ Matm,n(K), A≠0

Teorema degli Orlati (Teorema di Kronecker)

A ∈ Matm,n(K).
Il rango di A è p ⇐⇒∃ in A un minore di ordine p (Mp) non singolare ∧ ogni minore di ordine p + 1 che contiene completamente Mp è singolare.
